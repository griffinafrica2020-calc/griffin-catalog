🚀 خطوات النشر (مرة واحدة):
  1. افتح github.com وأنشئ مستودعاً جديداً (مثلاً: griffin-catalog)
  2. ارفع كل محتويات هذا المجلد كما هي (مع مجلد img)
  3. من Settings ← Pages ← Source: اختر main branch ثم Save
  4. بعد دقيقة يصبح الرابط:
     https://اسمك.github.io/griffin-catalog/

✅ ما ستحصل عليه تلقائياً:
  • تحميل أول أسرع بكثير (الصور تُحمّل عند الحاجة فقط)
  • عمل كامل بلا إنترنت بعد أول زيارة (Service Worker)
  • زر "إضافة إلى الشاشة الرئيسية" على أندرويد وآيفون
  • نتائج Lighthouse أعلى بكثير من النسخة المدمجة

🔁 عند تحديث الكتالوج مستقبلاً:
  استبدل index.html (وأي صور جديدة في img/) وارفعها،
  وغيّر رقم النسخة داخل sw.js من griffin-v1 إلى griffin-v2
  ليتلقى الزوار التحديث فوراً.
